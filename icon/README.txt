Thư mục icon ứng dụng Gia Phả
=============================

Đặt file icon của bạn vào đây với tên:

  main-icon.png   — icon chính (đang dùng)
  main-icon.ico   — bản .ico cho Windows (tùy chọn, ưu tiên hơn .png)

Hoặc tên thay thế: app.ico, app.png

Khi đóng gói .exe (PyInstaller), thêm:
  --icon icon/main-icon.ico
  (hoặc chuyển main-icon.png sang .ico trước)

Sau khi thêm file, khởi động lại: python main.py
